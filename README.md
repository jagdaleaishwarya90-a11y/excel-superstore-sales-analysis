# excel-superstore-sales-analysis(Excel project)
Excel dashboard project analyzing superstore sales data using pivot tables, charts and slicers.
## Project Overview

This project analyzes Superstore sales data using Microsoft Excel. The goal is to understand sales performance,profit trends, and customer behavior.
## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
- Dashboard Design
## key Analysis
- Region-wise sales
- category wise sales
- Top customers by sales
- Monthly sales Trend
## Dashboard Features
- Interactive slicers
- Multiple charts
- Sales performance insights
   ## Insights
  - Some regions generate higher sales compared to others.
  - Technology category contribute the highest revenue.
  - A few customers contribute significantly to overall sales.
## Conclusion
The project demonstrates how excel can be used for data cleaning,analysis, and building interactive dashboard for business insights.
